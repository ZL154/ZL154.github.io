Open-source Jellyfin plugins

Five plugins
for self-hosted Jellyfin.

Open-source, server-side plugins by ZL154 that extend Jellyfin — sign-in security, star ratings with external sync, viewing achievements, missing-episode scanning and folder-based prerolls.

5 plugins maintained C# · server-side MIT & open source

The suite

01 — 05
  1. 01

    JellyfinSecurity↗

    Hardens Jellyfin sign-in: native two-factor auth (TOTP & email), passkeys / WebAuthn, OIDC / SSO for Google, Authentik, Authelia and Keycloak, brute-force IP banning and impossible-travel detection.

    two-factor authpasskeysoidc / ssobrute-force defense

    GitHub · Release · Docs

  2. 02

    StarTrack↗

    Letterboxd-style star ratings with two-way sync to Trakt, Simkl, Letterboxd and Yamtrack — plus watched / liked sync and a backfill of every rating you've already given.

    star ratingstwo-way synctrakt · simkl · letterboxdbackfill

    GitHub · Release

    Details →
  3. 03

    AchievementBadges↗

    Achievement-style badges earned from your viewing and reading activity — first watches, binges, late-night sessions — with a backfill that reads your existing history.

    badgesviewing & readingmilestonesbackfill

    GitHub · Release

  4. 04

    MissingEpisodes↗

    Scans your library against Sonarr to find the episodes you're missing — and can kick off an automatic search to fill the gaps.

    Projectionist↗

Built in the open, run on real servers.

Each plugin starts from something I wanted on my own Jellyfin instance and ships once it survives daily use. Issues and pull requests are welcome — and if any of these earn a place in your setup, you can keep them going through Ko-fi or GitHub Sponsors.
